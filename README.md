# Optimal Charger Location for Electric SOVs in Offshore Wind Farms

This repository contains the files required to reproduce the results presented in the paper:

**"Optimal Charger Location for Electric SOVs in Offshore Wind Farms"**
*Agus Hasan*

## Running the Simulations

The main script is:

`python/run_all.py`

Run this script to reproduce the numerical results presented in the paper.

## Generating the Figures

To generate all figures used in the paper, run:

`make_all_figures.py`
